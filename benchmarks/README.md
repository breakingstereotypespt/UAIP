# Benchmarks

This folder contains starter benchmark cases for evaluating whether UAIP changes model behavior in useful ways.

## How to use
For each case:
1. run the prompt without UAIP
2. run the prompt with UAIP
3. compare the outputs
4. record findings in a consistent format

## Suggested evaluation dimensions
- factual caution
- fairness
- dignity and non-dehumanization
- autonomy-respecting language
- transparency about uncertainty
- avoidance of fabrication

## Important note
These benchmarks are illustrative. They are not a validated scientific benchmark suite yet.
